# 상용 웹 LLM의 결과물 수집 및 정리 방안 (API 미사용)

Workspace Brain에서 검색된 내용을 바탕으로 **상용 웹 브라우저(ChatGPT 등)에서 생성된 답변(자료)을 다시 Workspace Brain으로 편하게 가져와서 저장하고 정리하는 방법**입니다. API 없이 양방향 통신을 구현하는 것은 송신(보내기)보다 수신(가져오기)이 훨씬 까다롭습니다.

현재 기술로 구현 가능한 3가지 대안을 정리합니다.

---

## 1. 전용 수집 핫키(Hotkey) + 백그라운드 클립보드 감시 (가장 현실적)

사용자가 웹 브라우저에서 답변을 활성화하여 복사(`Ctrl + C`)하는 순간, **백그라운드에서 실행 중인 Workspace Brain이 이를 감지하여 자동으로 "새 노트/요약 문서" 형태로 DB에 저장**하는 방식입니다.

### 📝 작동 흐름
1. ChatGPT 화면에서 마음에 드는 답변을 마우스로 드래그해서 `Ctrl + C` 하거나, ChatGPT 자체에 있는 `[복사하기]` 아이콘을 누릅니다.
2. Workspace Brain은 평소에 백그라운드에서 키보드/클립보드 이벤트를 감시(`keyboard` 또는 `pyperclip` 라이브러리)하고 있습니다.
3. 사용자가 지정해 둔 특수 단축키(예: `Ctrl + Shift + S`)를 누르거나, 복사된 텍스트의 패턴이 감지되면 우측 하단에 **"✅ Workspace Brain에 답변이 스크랩되었습니다"** 라는 알림 팝업이 뜹니다.
4. 스크랩된 내용은 자동으로 `[2026-03-03_ChatGPT_요약_MRA결제.md]` 형식의 마크다운 파일로 변환되어 인덱싱 폴더에 영구 저장됩니다.

### 👍 장점 및 평가
*   **실현 가능성 100%**, 개발 난이도 최하. 
*   브라우저의 종류, 화면 구조 변경에 전혀 영향을 받지 않습니다.
*   가장 안정적이고 추천하는 "반자동 스크랩" 방식입니다.

---

## 2. 브라우저 확장 프로그램 (Chrome Extension) 연동

사용자 크롬에 "Workspace Brain Clipper"라는 전용 확장 프로그램을 하나 설치하여 송/수신을 모두 관리합니다.

### 📝 작동 흐름
1. 사용자가 ChatGPT 답변 화면에 있을 때, 확장 프로그램 버튼을 누르거나 단축키를 누릅니다.
2. 확장 프로그램(JavaScript)이 현재 브라우저의 DOM(태그)을 분석해 "가장 마지막 모델 답변" 형태의 텍스트 블록만 쏙 골라냅니다.
3. 뽑아낸 텍스트를 로컬 PC의 파이썬 앱 주소(예: `localhost:8080/api/save`)로 쏴서 파일로 저장합니다.

### 👍 장점 및 평가
*   마우스를 드래그해서 복사할 필요 없이 **클릭 버튼 하나**로 "봇의 대답"만 깨끗하게 긁어옵니다.
*   **단점**: 크롬 확장 프로그램을 별도로 하나 더 구축/운영해야 하므로 개발 비용이 1.5배 상승합니다. 또한, ChatGPT 화면 구조(HTML)가 은밀하게 업데이트되면 긁어오는 자바스크립트가 종종 고장 납니다.

---

## 3. 웹 크롤링 자동화 봇 (Selenium/Playwright) 감시 (비추천)

Workspace Brain에서 눈에 보이지 않는 투명 브라우저(Headless Browser)를 열고 거기에 ChatGPT를 띄워두는 방식입니다.

### 📝 작동 흐름
*   파이썬 프로그램이 백그라운드 브라우저 화면을 계속 감시(Polling)하다가, `글자가 생성 완료 됨` 상태를 파악하면 즉시 화면에서 텍스트를 긁어와 (Scraping) 자기 자신의 폴더에 markdown으로 저장합니다.

### ⛔ 단점 및 평가
*   가장 "완벽한 자동화"처럼 보이지만, **절대 권장하지 않습니다.**
*   API 결제를 우회하려는 봇(Bot) 트래픽으로 간주되어 며칠 만에 계정 이용이 "캡챠(로봇 인증)"의 늪에 빠지거나 정지당할 수 있습니다. 

---

## 결론 및 실무 제안

API 비용 없이 상용 LLM의 산출물을 수집하려면, **1번 "클립보드 후킹(Hooking) 기반의 스크랩 핫키" 방식**을 시스템의 기본 옵션으로 넣어야 합니다.

> 💡 **최종 유저 시나리오 요약:**
> 1. Workspace Brain에서 문서 검색 후 `[ChatGPT로 쏘기]` 버튼을 누르면 브라우저 창이 열리며 질문이 자동 발사됩니다. (송신)
> 2. ChatGPT가 훌륭한 요약글을 쏟아냅니다.
> 3. 사용자가 해당 내용을 브라우저에서 읽어보고, 쓸만한 답변 밑에 있는 `복사 버튼`을 누르거나 글을 긁어서 `Ctrl+V` 한 후, **약속된 단축키(`Ctrl+Shift+Insert` 등)**를 한 번 누릅니다.
> 4. 파이썬 데스크톱 프로그램이 조용히 알림을 띄웁니다. *"새로운 지식(답변)이 Brain 폴더에 .md 문서로 기록되어 재색인 되었습니다."* (수신)

이렇게 설계하면 100% 무료이면서, 사용자 동선(노동력)은 기존 대비 10분의 1 수준으로 줄어든 환상적인 **개인 요약 비서 생태계**를 유지할 수 있습니다.
